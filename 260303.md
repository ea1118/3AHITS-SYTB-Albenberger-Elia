## Visual Studio Code auf Linux Installieren

GitHub auf Linux VM übertragen

>Konsole Öffnen

```sh
setxkbmap de
mkdir SYTB
cd SYTB
git clone https://github.com/ea1118/3AHITS-SYTB-Albenberger-Elia
cd 3AHITS-SYTB-Albenberger-Elia
../..
```

Visual Studio Code für Linux Downloaden auf https://code.visualstudio.com/


```sh
cd Downloads
sudo apt install ./code_1.109.5-1771531656_amd64.deb
..
```

Visual Studio Code öffnen
```sh
cd SYTB/3AHITS-SYTB-Albenberger-Elia
code .
```

Visual Studio Code Konfigurieren

>Toggle Panel (Ctrl + J)

```sh
git config --global user.name "Elia Albenberger"    
git config --global user.email "elia.albenberger@htl-braunau.at"
```
Somit sieht man Name und Email sobald man änderungen **commited**
